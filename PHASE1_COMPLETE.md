## üìù Usage Notes

### Dashboard Analytics
- Analytics section appears automatically on the dashboard
- Metrics update when you refresh or navigate back to dashboard

### Bulk Import
1. Go to "Add Elements" section
2. Click "Download CSV template" to get the format
3. Fill in your elements
4. Click "Choose CSV File" and select your file
5. Review preview and click "Import"

### Advanced Search
1. Go to "Add Elements" section
2. Use the search/filter panel at the top
3. Set your criteria
4. Results filter in real-time
5. Click "Save Filter Preset" to save for later

### Sortable Tables
1. Go to "Create Relationships" section
2. Click any column header to sort
3. Click again to reverse sort order
4. Visual indicators show current sort
